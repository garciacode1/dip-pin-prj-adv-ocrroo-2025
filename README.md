## Tesseract OCR requirement

This project uses pytesseract for OCR. The Tesseract OCR engine must also be installed on the computer.

On Windows, install Tesseract OCR and make sure it is available in the system PATH, or set the path 
to `tesseract.exe` in the Python code.
