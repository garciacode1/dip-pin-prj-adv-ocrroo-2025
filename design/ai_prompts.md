# AI Prompts Used for Design

This file records AI assistance used during the design stage of the OCR Video Reader project.

-------------------------------------------------------------------------------
## Prompt 1: Persona

Prompt used:

Help me create a persona for an OCR Video Reader app. The target user is a programming student
who learns from coding tutorial videos and needs to extract text from video frames.

How it was used:

The AI helped draft the persona structure, including the user background, goals, 
needs, frustrations, and how the project helps the user. I reviewed and edited the 
wording before adding it to the project.
----------------------------------------------------------------------------------
# Prompt 2: Wireframe Prompt

Prompt used:

Create a wireframe prompt for UX Pilot for a web application called OCR Video Reader.
The app should include video selection, timestamp input, frame preview, OCR result, 
validation messages, and accessibility options.

How it was used:

The AI helped write a prompt for UX Pilot. I used UX Pilot to generate the wireframe
and then added the screenshots to the design folder.
-------------------------------------------------------------------------------------
## Prompt 3: Accessibility and Personalisation

Prompt used:

What personalisation features should I include for the assessment? 
Should I use dark mode, text size, or high contrast mode?

How it was used:

The AI helped compare personalisation options. I decided to include 
accessibility options such as high contrast mode, text scale, and dark mode 
in the wireframe.
