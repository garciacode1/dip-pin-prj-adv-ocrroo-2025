# Persona

## Name

Benny Bell

## Role

Diploma of IT student

## Background

Benny is learning programming and regularly watches recorded coding lessons, demonstrations, and tutorial videos. He often needs to pause the video to read code, copy important text, or understand what is shown on the screen. This can take time, especially when the video quality is low or the text is small.

## Goals

Benny wants an easy-to-use tool that can take text from a selected video frame. He wants to choose a video, enter a time in seconds, view the frame, and get the extracted OCR text.

## Needs

* A simple and clear user interface
* A way to select a video time
* A preview of the selected video frame
* OCR text shown in an easy-to-read format
* Validation to stop invalid time entries
* Personalisation options, such as dark mode or larger text

## Frustrations

* Copying code from videos manually is slow
* Small or blurry text can be difficult to read
* Finding the correct timestamp can be annoying
* Complicated tools make the task harder than it should be

## How this project helps

The OCR Video Reader helps Benny by letting him select a time in a video, capture the frame, and extract the text using OCR. This makes it easier for him to study coding videos and collect useful code or notes.
