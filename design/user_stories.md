# User Stories

## User Story 1: Improve readability for visual impairment

As a student with visual impairment,  
I want to adjust the text size and use a high-contrast display mode,  
so that I can read the OCR results and use the interface more comfortably.

### Acceptance Criteria
- The user can increase the text size.
- The user can turn on a high-contrast or dark mode.
- The OCR results are displayed clearly and remain readable after the setting is changed.

## User Story 2: Support learning without audio

As a student who cannot hear well or cannot hear audio,
I want to extract text from programming tutorial videos,
so that I can understand the code and lesson content without depending only on sound.

### Acceptance Criteria

* The user can select a video frame from a tutorial.
* The system can extract visible code or text from the selected frame.
* The OCR text is shown clearly on the page.
* The user can copy the extracted text for study notes.

## User Story 3: Select a video frame

As a programming student,  
I want to enter a time in seconds for a video,  
so that I can view the exact frame I need to study.

### Acceptance Criteria
- The user can enter a time in seconds.
- The system checks that the time is valid.
- The system shows the selected video frame.
